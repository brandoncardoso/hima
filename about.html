About

Hima is a Jekyll theme. You can find out more about customizing your Jekyll site, as well as basic Jekyll usage documentation on the official Jekyll website. You can also find the source code for Jekyll on GitHub.

Information about this theme, how to customize it, and its source code are all available on the Hima GitHub repository.
